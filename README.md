# Bank Marketing EDA Using Python

## Project Overview

This project analyzes bank marketing campaign data using Python to identify customer behavior, subscription trends, and campaign effectiveness through exploratory data analysis (EDA).

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Tasks Performed

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Customer Segmentation
- Trend Analysis

## Insights

- Identified factors influencing customer subscription to term deposit campaigns
- Analyzed customer demographics and campaign response patterns
- Explored relationships between call duration and subscription outcomes

## Files Included

- bank-marketing-data.csv
- bank-marketing-eda.ipynb
